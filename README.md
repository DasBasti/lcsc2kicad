# JLCPCB/LCSC SMT Parts library to KiCad Library Converter

This script will fetch the current available parts for SMT 
assembly offered by JLCPCB and generate KiCad Part Library 
files.

# Setup
    pip install xlrd

# Run
    python generate.py
